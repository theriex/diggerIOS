# diggerIOS

Like it says on the tin.  This is Digger on IOS.  For more on Digger go to https://diggerhub.com or https://github.com/theriex/digger

To build:
Create new Xcode project
Product Name: diggerIOS
Team: Your team e.g. Eric Parker (Personal Team)
Interface: Storyboard
Language: Swift
Include Tests is selected by default, left as is.
Select iOS platform, then app.
Create project in your development directory.  Setup makes
diggerIOS/
  diggerIOS
  diggerIOS.xcodeproj
  diggerIOSTests
  diggerIOSUITests

Move github source files into xcode created directories.  Tether your target IOS device and run.
